# Java Bean

## 实体类

编写套路：

1. 定义实例字段，基本类型值用包装类（因为可`null`）
2. 对枚举性的字段，用`static final`定义具名常量，根据外部是否要用写`private`或`public`
3. 自动生成构造方法，常用三种：
    1. 无参
    2. 除了ID外的所有参数
    3. 所有参数
4. 自动生成所有字段的getter和setter方法
5. 自动生成`toString()`方法
6. 对枚举性的字段，可添加一些`isXxx()`判断方法，判断是否是某个枚举值
7. 对有默认值的字段，可添加一些`setXxxValue()`方法，设置默认值

``` java
/**
 * XXX实体类
 */
public class Entity {
    // ------------------------------------------------------------------------
    // 枚举常量定义
    // ------------------------------------------------------------------------

    /**
     * 类型：A
     */
    public static final String TYPE_A = "A";

    // ------------------------------------------------------------------------
    // 实例字段定义
    // ------------------------------------------------------------------------

    /**
     * ID
     */
    private Integer id;
    /**
     * 类型
     */
    private String type;
    /**
     * 创建时间
     */
    private Date createTime;

    // ------------------------------------------------------------------------
    // 构造方法定义（自动生成）
    // ------------------------------------------------------------------------

    // 构造方法：无参
    public Entity() {
    }

    // 构造方法：除ID外的所有参数
    public Entity(String type) {
        this.type = type;
    }

    // 构造方法：所有参数
    public Entity(Integer id, String type) {
        this.id = id;
        this.type = type;
    }

    // ------------------------------------------------------------------------
    // 判断方法定义
    // ------------------------------------------------------------------------

    public boolean isTypeA() {
        return TYPE_A.equals(type);
    }

    // ------------------------------------------------------------------------
    // 设置默认值方法定义
    // ------------------------------------------------------------------------

    public void setCreateTimeNow() {
        this.createTime = new Date();
    }

    // ------------------------------------------------------------------------
    // getter和setter方法定义（自动生成）
    // ------------------------------------------------------------------------

    // getter方法
    public Integer getId() {
        return id;
    }

    // setter方法
    public void setId(Integer id) {
        this.id = id;
    }

    // 更多getter和setter方法

    // ------------------------------------------------------------------------
    // toString()方法定义
    // ------------------------------------------------------------------------

    @Override
    public String toString() {
        return "Entity{" +
            "id=" + id +
            ", type='" + type + '\'' +
            ", createTime='" + createTime + '\'' +
            '}';
    }
}
```
